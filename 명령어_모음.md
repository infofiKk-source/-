# Firebase 설치 명령어 모음

## ✅ 올바른 명령어 (복사해서 사용하세요)

### 1. 프로젝트 폴더로 이동
```powershell
cd "C:\Users\명재민\Desktop\위로 플랫폼"
```

### 2. Firebase 설치
```powershell
npm install firebase
```

---

## ⚠️ 주의사항

- 경로에 공백이 있으면 **반드시 따옴표로 감싸야 합니다**
- `cd "C:\Users\명재민\Desktop\위로 플랫폼"` ✅ (따옴표 있음)
- `cd C:\Users\명재민\Desktop\위로 플랫폼` ❌ (따옴표 없음 - 오류 발생)

---

## 📝 한 번에 실행하기

터미널에 아래 명령어를 **한 줄씩** 복사해서 붙여넣으세요:

```powershell
cd "C:\Users\명재민\Desktop\위로 플랫폼"
```

그 다음:

```powershell
npm install firebase
```

---

## 🔍 현재 위치 확인

현재 어느 폴더에 있는지 확인하려면:

```powershell
pwd
```

또는

```powershell
Get-Location
```

---

## 💡 팁

VS Code에서 터미널을 열면 자동으로 프로젝트 폴더로 이동되어 있을 수 있습니다.
그 경우 바로 `npm install firebase`만 실행하면 됩니다!
