# Vercel 반영 안 되는 문제 해결

## 🔍 확인 사항

### 1. 로컬에서 확인
브라우저에서 `http://localhost:3000` 접속:
- 개발 서버가 실행 중인지 확인
- 브라우저 하드 리프레시: `Ctrl + Shift + R`
- `/post/new` 페이지에서 링크 입력 필드가 보이는지 확인

### 2. Git 푸시 확인
터미널에서 실행:
```powershell
git status
```

변경사항이 있으면:
```powershell
git add .
git commit -m "게시글에 링크 추가 기능 구현"
git push
```

### 3. Vercel 배포 상태 확인
1. https://vercel.com/dashboard 접속
2. 프로젝트 클릭
3. **"Deployments"** 탭 확인:
   - 🟡 노란색 점: 배포 중 (1-3분 대기)
   - 🟢 초록색 체크: 배포 완료
   - 🔴 빨간색 X: 배포 실패 (에러 확인)

### 4. 배포 실패 시
1. 실패한 배포 클릭
2. **"Build Logs"** 확인
3. 에러 메시지 확인

### 5. 브라우저 캐시 문제
Vercel 사이트에서:
- `Ctrl + Shift + R` (하드 리프레시)
- 또는 시크릿 모드에서 확인

---

## 🛠️ 빠른 해결 방법

### 방법 1: 로컬에서 먼저 테스트
```powershell
npm run dev
```
브라우저에서 `http://localhost:3000/post/new` 접속
→ 링크 입력 필드가 보이는지 확인

### 방법 2: Git 푸시 다시 확인
```powershell
git status
git add .
git commit -m "게시글 링크 기능 추가"
git push
```

### 방법 3: Vercel 수동 재배포
1. Vercel 대시보드 접속
2. 프로젝트 > **"Deployments"** 탭
3. 최신 배포 옆 **"..."** 메뉴 클릭
4. **"Redeploy"** 클릭

---

## ❓ 어떤 부분이 안 되나요?

1. **로컬에서 안 보임**: 브라우저 새로고침 (`Ctrl + Shift + R`)
2. **Vercel에서 안 보임**: 배포 상태 확인 및 재배포
3. **링크 입력 필드가 안 보임**: 개발 서버 재시작
4. **링크가 저장 안 됨**: Firebase 콘솔에서 확인
