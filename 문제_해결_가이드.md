# 게시물 올리기 문제 해결 가이드

## 🔍 문제 확인

`lib/firebase/config.ts` 파일을 보니 Firebase 설정 정보가 아직 입력되지 않았습니다!

현재 상태:
```typescript
apiKey: "여기에-apiKey-입력",  // ❌ 아직 입력 안 됨
authDomain: "여기에-authDomain-입력",  // ❌ 아직 입력 안 됨
```

## ✅ 해결 방법

### 1단계: Firebase 콘솔에서 설정 정보 가져오기

1. 브라우저에서 https://console.firebase.google.com 접속
2. 프로젝트 선택
3. 왼쪽 상단 **톱니바퀴 아이콘** ⚙️ 클릭
4. **"프로젝트 설정"** 클릭
5. 페이지 중간 **"내 앱"** 섹션 찾기
6. 웹 앱이 없으면 **`</>` 아이콘** 클릭하여 추가
7. 나타나는 코드에서 다음 정보 복사:

```javascript
const firebaseConfig = {
  apiKey: "AIzaSy...",  // 이 값 복사
  authDomain: "...",    // 이 값 복사
  projectId: "...",     // 이 값 복사
  storageBucket: "...", // 이 값 복사
  messagingSenderId: "...", // 이 값 복사
  appId: "..."         // 이 값 복사
};
```

### 2단계: 설정 정보 입력하기

1. VS Code에서 `lib/firebase/config.ts` 파일 열기
2. 복사한 값을 아래처럼 입력:

```typescript
const firebaseConfig = {
  apiKey: "AIzaSyAbc123...",  // 여기에 붙여넣기
  authDomain: "my-project.firebaseapp.com",  // 여기에 붙여넣기
  projectId: "my-project-id",  // 여기에 붙여넣기
  storageBucket: "my-project.appspot.com",  // 여기에 붙여넣기
  messagingSenderId: "123456789",  // 여기에 붙여넣기
  appId: "1:123456789:web:abcdef",  // 여기에 붙여넣기
}
```

3. **저장** (`Ctrl + S`)

### 3단계: 확인 사항

#### ✅ Authentication 활성화 확인
1. Firebase 콘솔 > **Authentication** 클릭
2. **Sign-in method** 탭 클릭
3. **"익명"** 또는 **"Anonymous"** 찾기
4. **"사용 설정"** 토글이 **ON**인지 확인
5. OFF면 ON으로 변경 후 **"저장"**

#### ✅ Firestore Database 확인
1. Firebase 콘솔 > **Firestore Database** 클릭
2. 데이터베이스가 생성되어 있는지 확인
3. 없으면 **"데이터베이스 만들기"** 클릭

#### ✅ Firestore 보안 규칙 확인
1. **"규칙"** 탭 클릭
2. 다음 규칙이 있는지 확인:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /posts/{postId} {
      allow read: if true;
      allow create: if request.auth != null;
      allow update, delete: if false;
    }
    match /comments/{commentId} {
      allow read: if true;
      allow create: if request.auth != null;
      allow update, delete: if false;
    }
    match /reactions/{reactionId} {
      allow read: if true;
      allow create: if request.auth != null;
      allow update, delete: if false;
    }
  }
}
```

3. 없으면 추가하고 **"게시"** 클릭

### 4단계: 브라우저 콘솔 확인

1. 브라우저에서 `http://localhost:3000` 열기
2. **F12** 눌러서 개발자 도구 열기
3. **Console** 탭 클릭
4. 게시물 올리기 시도
5. 빨간색 에러 메시지 확인

**자주 보는 에러:**
- `auth/network-request-failed`: 네트워크 문제 또는 설정 정보 오류
- `permission-denied`: Firestore 보안 규칙 문제
- `auth/configuration-not-found`: Firebase 설정 정보가 잘못됨

## 🆘 여전히 안 되면

1. **브라우저 콘솔 에러 메시지** 확인
2. **화면에 표시된 에러 메시지** 확인
3. **Firebase 콘솔에서 설정이 올바른지** 다시 확인
