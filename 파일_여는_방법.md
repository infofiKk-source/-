# VS Code에서 파일 여는 방법 (초보자용)

## 방법 1: 왼쪽 파일 탐색기 사용 (가장 쉬움)

### 1단계: VS Code 왼쪽 사이드바 확인
1. VS Code를 엽니다
2. 화면 왼쪽에 파일 목록이 보이는지 확인
3. 안 보이면:
   - 상단 메뉴: **View** (보기) 클릭
   - **Explorer** (탐색기) 클릭
   - 또는 `Ctrl + Shift + E` 키 누르기

### 2단계: 폴더 구조 찾기
왼쪽 파일 목록에서 다음 순서로 찾기:

```
📁 위로 플랫폼 (프로젝트 루트)
  📁 lib
    📁 firebase
      📄 config.ts  ← 이 파일!
```

### 3단계: 파일 클릭
- `config.ts` 파일을 **마우스 왼쪽 버튼으로 클릭**
- 파일이 오른쪽 편집 영역에 열립니다!

---

## 방법 2: 검색으로 찾기

### 1단계: 파일 검색 열기
- `Ctrl + P` 키 누르기
- 또는 상단 메뉴: **File** > **Open File...**

### 2단계: 파일 이름 입력
검색창에 입력:
```
config.ts
```

### 3단계: 파일 선택
- `lib/firebase/config.ts` 파일이 나타나면
- **Enter** 키 누르기 또는 **마우스로 클릭**

---

## 방법 3: 경로로 직접 열기

### 1단계: 파일 열기 메뉴
- 상단 메뉴: **File** (파일) 클릭
- **Open File...** (파일 열기) 클릭
- 또는 `Ctrl + O` 키 누르기

### 2단계: 경로 입력
파일 탐색기 창이 열리면:
1. 주소창에 경로 입력:
   ```
   C:\Users\명재민\Desktop\위로 플랫폼\lib\firebase\config.ts
   ```
2. **Enter** 키 누르기

## 📝 파일을 열었으면

파일 내용이 이렇게 보일 것입니다:

```typescript
import { initializeApp, getApps } from "firebase/app"
import { getAuth } from "firebase/auth"
import { getFirestore } from "firebase/firestore"

// Firebase 설정 정보
// Firebase 콘솔 > 프로젝트 설정 > 일반 > 내 앱 > 웹 앱의 설정 정보를 여기에 입력하세요
const firebaseConfig = {
  apiKey: "여기에-apiKey-입력",
  authDomain: "여기에-authDomain-입력",
  projectId: "여기에-projectId-입력",
  storageBucket: "여기에-storageBucket-입력",
  messagingSenderId: "여기에-messagingSenderId-입력",
  appId: "여기에-appId-입력",
}
```

이제 이 부분을 수정하면 됩니다!

---

## 💡 팁

- 파일을 열면 VS Code 상단 탭에 파일 이름이 표시됩니다
- 여러 파일을 열면 탭으로 전환할 수 있습니다
- 파일을 수정하면 탭에 점(·)이 표시되어 저장되지 않았음을 알려줍니다
